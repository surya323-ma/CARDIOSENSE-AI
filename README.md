## 🧠 Overview

**CardioSense AI** is an advanced Machine Learning-powered web application that predicts cardiovascular disease risk based on patient clinical data.

Built with a sleek **Crimson UI**, this system provides:
- 📊 Real-time risk analysis  
- 🧬 Clinical insights  
- 📈 Data visualization  
- 📋 Patient report generation  

---

## ⚙️ Tech Stack

| Layer | Technology |
|------|-----------|
| Frontend | Streamlit |
| ML Model | K-Nearest Neighbors (KNN) |
| Scaling | StandardScaler |
| Visualization | Plotly |
| Language | Python |

---

## 🧬 Features

✨ **AI Diagnostic Engine**  
Predicts heart disease severity using 13 clinical features  

📊 **Interactive Dashboard**  
- Radar charts  
- Risk distribution graphs  
- Simulation analysis  

🧠 **Model Insights Panel**  
Explains ML logic & feature importance  

📋 **Automated Patient Report**  
Generates structured clinical summaries  

🎨 **Modern UI/UX**  
Custom CSS + Glassmorphism + Animations

## 🚀 How to Run

### 1️⃣ Clone Repository
git clone https://github.com/surya323-ma/CARDIOSENSE-AI.git
cd CARDIOSENSE-AI
## 2️⃣ Install Dependencies
pip install -r requirements.txt

(If no requirements.txt, install manually:)

pip install streamlit scikit-learn pandas numpy plotly
## 3️⃣ Run App
streamlit run app.py
